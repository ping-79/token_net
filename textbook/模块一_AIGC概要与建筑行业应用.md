## 第1课时 认识AIGC

### 章节导读

在过去的几百年里，建筑设计的表达方式经历了深刻的变革。清代皇家建筑世家"样式雷"家族，用精巧的"烫样"（立体模型）向皇帝呈现建筑方案，这是中国古代建筑师用技术手段辅助设计表达的杰出典范。如今，人工智能生成内容（AIGC）技术正在开启建筑设计表达的新纪元——我们可以用一段文字描述，让AI在几秒钟内生成建筑效果图、室内渲染图、景观意向图，甚至生成方案展示视频。

本课时将带你走进AIGC的世界，理解这项技术的本质，了解它的发展历程，以及它将如何深刻改变建筑行业的工作方式。

### 一、AIGC是什么？

#### 1.1 从三个缩写说起

在正式学习之前，我们先厘清几个容易混淆的概念：

| 缩写 | 全称 | 含义 | 举例 |
|------|------|------|------|
| **AI** | Artificial Intelligence（人工智能） | 让机器具备类似人类智能的技术总称 | 人脸识别、语音助手、自动驾驶 |
| **GC** | Generated Content（生成内容） | 由某种方式生成的内容 | — |
| **AIGC** | AI Generated Content（人工智能生成内容） | 由AI自动生成的文字、图片、音频、视频等内容 | ChatGPT写文章、Midjourney画图 |

与AIGC相关的还有两个概念需要了解：

- **PGC**（Professionally Generated Content）：专业生成内容，如建筑设计师绘制的施工图
- **UGC**（User Generated Content）：用户生成内容，如业主在社交媒体发布的装修照片

**AIGC的本质**：让AI成为内容的"创作者"。你只需要告诉AI你想要什么（输入指令），AI就能自动生成相应的内容（输出结果）。

#### 1.2 生成式AI vs 传统AI

为了更好地理解AIGC，我们需要区分两种AI：

| 对比维度 | 传统AI（分析式AI） | 生成式AI（AIGC） |
|---------|-------------------|-----------------|
| **核心能力** | 识别、分类、预测 | 创造、生成、设计 |
| **工作方式** | 从数据中找规律，给出判断 | 学习大量数据后，创造新内容 |
| **输入→输出** | 图片→"这是一栋住宅" | "一栋现代风格住宅"→生成效果图 |
| **建筑行业例子** | 结构安全检测、能耗预测 | 生成效果图、撰写设计说明 |
| **类比** | 像一个"鉴定师" | 像一个"创作者" |

> **课堂思考**：你平时使用的手机功能中，哪些属于传统AI（如人脸解锁），哪些属于生成式AI（如AI修图）？

#### 1.3 AIGC能生成什么？

AIGC的"生成"能力覆盖了几乎所有内容形式：

| 内容类型 | 说明 | 建筑行业应用举例 |
|---------|------|-----------------|
| **文字** | 文章、报告、代码等 | 设计说明书、可行性报告、规范解读 |
| **图像** | 照片、插画、效果图等 | 建筑效果图、室内渲染图、景观意向图 |
| **音频** | 语音、音乐、音效等 | 方案汇报配音、项目宣传配乐 |
| **视频** | 短片、动画、特效等 | 建筑漫游动画、项目宣传片 |
| **代码** | 程序、网页、应用等 | 项目展示网页、数据分析工具 |
| **3D模型** | 三维模型（发展中） | AI辅助建筑建模（新兴方向） |

#### 2.1 从规则到智能：AI发展简史

```

[规则时代]     →     [机器学习时代]    →   [深度学习时代]   →   [大模型时代]
人工编写规则         让机器从数据中学习     神经网络突破          GPT、多模态、Agent
专家系统             图像识别、推荐系统     AlphaGo战胜人类       AIGC全面爆发
```

| 时间 | 事件 | 意义 |

#### 2.3 AIGC技术的基本原理（通俗版）

AIGC的核心技术是**大语言模型（LLM）**和**扩散模型（Diffusion Model）**，它们的工作原理可以用以下类比来理解：

**文字生成的原理——"超级联想"**

想象一个读过海量书籍的人，当你说出一句话的开头，他能根据经验推测最可能的下一个词：

```
输入："今天天气..."
AI推测 →"今天天气真好"（最可能）
        "今天天气不错"（较可能）
        "今天天气转阴"（可能）
```

大语言模型就是这样一个"超级联想机器"，它通过学习互联网上的海量文本，掌握了语言的规律，能够持续预测并生成连贯的文字内容。

**图像生成的原理——"从噪声中雕刻"**

扩散模型的工作方式可以类比为雕塑：

```
一块随机的噪点图（石头）
    ↓ 根据你的文字描述（设计图纸）
    ↓ 一步步去除噪点（凿掉多余的石头）
    ↓ 逐渐显现出清晰的图像（完成雕塑）
生成最终的效果图
```

> **注意**：AI并不真正"理解"建筑设计，它是通过学习大量建筑图片的模式和特征，来生成看起来合理的新图像。这就是为什么AI有时会生成不符合实际的内容（我们将在模块二第6课时详细讨论"AI幻觉"问题）。

#### 3.2 行业层面：建筑行业的数字化转型

建筑行业正在经历深刻的数字化转型，AIGC是其中的关键推动力：

**传统工作流程 vs AIGC赋能的工作流程**

| 阶段 | 传统方式 | AIGC赋能后 |
|------|---------|-----------|
| **方案构思** | 手绘草图→反复修改→数日完成 | AI快速生成多个方案→分钟级完成 |
| **效果图制作** | 3D建模→渲染→后期→数天 | 文字描述/草图→AI生成→数分钟 |
| **设计说明** | 人工撰写→反复修改→数小时 | AI辅助生成→人工审核→数十分钟 |
| **方案汇报** | 制作PPT→录制视频→数天 | AI一键生成PPT和视频→数小时 |
| **规范查询** | 翻阅规范文件→人工检索 | AI知识库→智能问答→秒级响应 |

#### 3.3 个人层面：职业发展的新要求

> **课堂讨论**：面对AI的冲击，建筑行业从业者应该感到焦虑还是兴奋？

**AIGC对建筑从业者的影响可以概括为"三个转变"：**

1. **工具转变**：从手动操作到AI辅助，效率大幅提升
2. **能力转变**：从"会画图"到"会指挥AI画图"，提示词能力成为新技能
3. **角色转变**：从"执行者"到"决策者"，人的核心价值在于审美、判断和创新

**建筑专业学生应具备的AIGC素养：**
- 能熟练使用AI工具辅助专业学习和工作
- 能编写有效的提示词，控制AI输出质量
- 能判断AI生成内容的准确性和专业性
- 了解AI的局限性和使用边界
- 具备AI伦理意识和法律常识

### 四、如何学习AIGC

#### 4.1 学习AIGC的正确姿势

学习AIGC不同于传统的软件学习，需要注意以下几点：

**1. "用"比"学"更重要**

AIGC工具的操作并不复杂，关键是在实际场景中反复使用。就像学游泳，看再多理论不如下水实践。

**2. 保持持续学习的心态**

AIGC技术发展极快，工具和功能几乎每月都在更新。今天的"最佳工具"可能半年后就被新工具取代。重要的是掌握学习方法，而不是死记某个工具的操作步骤。

**3. 结合专业场景学习**

不要为了学AI而学AI，而要思考"这个AI工具能帮我解决什么专业问题"。本教材的每个模块都围绕建筑行业的实际场景展开。

**4. 学会提问比学会回答更重要**

与AI交互的核心是"提问"。一个好的提示词（Prompt）往往比选择哪个AI工具更重要。

#### 4.2 推荐的学习路径

```
入门阶段：文字对话 → 学会与AI交流，掌握提示词技巧
   ↓
进阶阶段：图像生成 → 用AI辅助建筑设计可视化
   ↓
提升阶段：音视频+PPT → 用AI制作完整的方案展示
   ↓
高阶阶段：智能体+编程 → 搭建个人AI工作流，提升综合效率
```

> 本教材正是按照这个学习路径编排的。

## 第2课时 AIGC与建筑行业

### 章节导读

上一课时我们认识了AIGC的基本概念。本课时将聚焦建筑行业，全面了解AIGC在建筑全生命周期中的应用场景，并掌握主流AIGC工具的分类和获取方式。

> **从"样式雷"说起**：清代雷氏家族七代人主持皇家建筑设计长达200余年，留下了大量珍贵的建筑图档和"烫样"。"烫样"是用纸板、秫秸、木头等材料制作的建筑立体模型，可以层层拆卸，展示建筑内部结构——这是古代建筑师向决策者展示设计方案的"可视化工具"。今天，AIGC让我们能用文字描述瞬间生成逼真的效果图和漫游视频，本质上与"样式雷"的烫样一样，都是用当时最先进的技术手段来更好地表达设计意图。从烫样到AIGC，变的是技术手段，不变的是中国建筑师对卓越表达的追求。

### 一、建筑行业AIGC应用全景图

AIGC技术可以贯穿建筑项目的全生命周期，在每个阶段都有具体的应用场景：

#### 1.1 规划阶段

| 应用方向 | 具体场景 | 涉及专业 |
|---------|---------|---------|
| AI场地分析 | 输入地块信息，AI分析日照、风环境、交通等 | 城乡规划 |
| AI方案生成 | 输入设计条件，AI自动生成多个布局方案 | 城乡规划、建筑设计 |
| AI政策解读 | AI辅助解读规划政策和设计条件 | 城乡规划 |
| AI数据可视化 | AI辅助制作规划分析图和数据图表 | 城乡规划 |

#### 1.2 设计阶段

| 应用方向 | 具体场景 | 涉及专业 |
|---------|---------|---------|
| AI效果图 | 文字/草图生成建筑效果图 | 建筑设计、室内设计、园林景观 |
| AI方案对比 | 快速生成多风格方案供业主选择 | 室内设计、建筑设计 |
| AI设计说明 | 自动生成设计说明书等文本 | 所有专业 |
| AI材料选型 | AI推荐材料搭配方案 | 室内设计、园林景观 |

#### 1.3 施工阶段

| 应用方向 | 具体场景 | 涉及专业 |
|---------|---------|---------|
| AI安全监测 | AI识别施工现场安全隐患 | 建筑设计（安全管理） |
| AI进度管理 | AI辅助分析施工进度数据 | 建筑设计 |
| AI安全培训 | AI生成安全培训图文和视频 | 所有专业 |

#### 1.4 运维阶段

| 应用方向 | 具体场景 | 涉及专业 |
|---------|---------|---------|
| AI智慧物业 | 智能客服、设备巡检报告 | 园林景观（物业管理） |
| AI能耗管理 | 分析建筑能耗数据，优化运行策略 | 建筑设计 |

### 二、四大专业方向的AIGC应用重点

根据人居环境与艺术学院的四个专业方向，AIGC的应用侧重各有不同：

#### 2.1 城乡规划方向

```
核心应用：
├── AI辅助场地分析与方案生成
├── AI生成规划效果图（鸟瞰图、街景图）
├── AI撰写规划分析报告
├── AI制作规划方案汇报材料
└── AI数据分析与可视化
```

**典型场景**：接到一个乡村振兴规划项目，用AI快速分析场地现状、生成多个布局方案的效果图、撰写规划说明书、制作汇报PPT和视频。

#### 2.2 建筑设计方向

```
核心应用：
├── AI生成建筑方案效果图
├── 手绘草图/白模 → AI渲染
├── AI撰写设计说明书
├── AI辅助规范查询
└── AI生成施工安全培训材料
```

**典型场景**：在方案初期，用文字描述建筑风格和功能需求，AI快速生成多个方案效果图供讨论；将手绘草图上传，AI生成精美的渲染效果图。

#### 2.3 室内设计方向

```
核心应用：
├── AI生成不同风格室内效果图
├── AI辅助方案对比与业主沟通
├── AI生成材料搭配方案
├── AI撰写室内设计说明
└── AI制作方案展示视频
```

**典型场景**：业主说"我想要北欧风格的客厅"，用AI在1分钟内生成10种北欧风格客厅效果图，业主当场选择喜欢的方向，大幅缩短沟通成本。

#### 2.4 园林景观方向

```
核心应用：
├── AI生成景观意向图和效果图
├── AI生成景观节点效果图
├── AI植物配置方案建议
├── AI撰写景观设计文本
└── AI生成项目宣传视频
```

**典型场景**：为一个城市公园项目生成不同季节、不同时段的景观效果图，用AI制作四季变换的景观漫游视频。

### 知识拓展：样式雷与中国建筑智慧

"样式雷"是清代雷氏建筑世家的美称。从康熙年间的雷发达开始，雷氏家族七代人主持皇家建筑设计，参与了故宫、天坛、圆明园、颐和园、清东陵、清西陵等重大工程的设计与营建。

雷氏家族最具特色的贡献是创造了"烫样"——一种用硬纸板、秫秸杆、木头等材料制作的建筑立体模型。烫样可以层层拆卸，展示建筑的外观、内部结构和周围环境，是向皇帝呈报设计方案的重要工具。

2007年，"样式雷"建筑图档被联合国教科文组织列入《世界记忆名录》，成为中国建筑文化的世界级遗产。

**从"烫样"到AIGC的启示**：

| 维度 | 样式雷的烫样 | 今天的AIGC |
|------|------------|-----------|
| **目的** | 让决策者直观理解设计方案 | 让客户和团队快速看到设计效果 |
| **技术** | 纸板、木材、手工制作 | 算法、大模型、自动生成 |
| **效率** | 数周制作一个烫样 | 数分钟生成一张效果图 |
| **精神** | 精益求精、世代传承 | 持续创新、技术迭代 |
| **共同点** | 都是用当时最先进的技术手段，辅助建筑设计的表达与沟通 |
