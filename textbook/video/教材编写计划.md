## 一、授课范围策略：「学院核心 + 全校选修」双轨

### 分两步走

**第一步（本学期）：先聚焦本学院**
- 面向人居环境与艺术学院的4个专业方向
- 案例全部围绕城乡规划、建筑设计、室内设计、园林景观
- 教材以建筑行业场景为主线，内容深度适中

**第二步（后续）：扩展为全校通识选修课**
- 在本学院版基础上增加通用案例章节
- 每个模块增加1-2个跨专业案例
- 教材设计为"核心内容 + 专业拓展包"结构

## 四、各模块详细内容大纲

### 模块一：AIGC概要与建筑行业应用（2课时）

**第1课时：认识AIGC**
- AIGC是什么？（生成式AI vs 传统AI）
- AIGC的发展历程（GPT→多模态→Agent）
- 国家政策：教育部AI赋能教育、住建部数字化转型

**第2课时：AIGC与建筑行业**
- 建筑行业AIGC应用全景图（规划/设计/施工/运维四阶段）
- 常用AIGC工具清单（文字类、图像类、视频类、综合类）
- 课堂互动：AI能解决哪些专业学习问题？

---

### 模块二：文字生成——建筑文本写作（4课时）

**第3课时：基础对话与学习辅助**
- AI对话的基本方法
- 实操：用AI学习建筑规范条文解读、翻译英文文献、分析设计案例

**第4课时：提示词工程**
- 提示词七要素：角色、背景、事项、要求、样式、风格、结构
- 建筑领域提示词模板库（设计说明、可研报告、投标书）

**第5课时：建筑专业文本生成实战**
- 实操：室内设计说明书 / 城乡规划报告 / 园林景观文本 / 施工方案

**第6课时：AI内容幻觉与信息验证**
- 什么是AI幻觉？建筑领域典型案例
- 减少幻觉的方法与验证技巧

---

### 模块三：图像生成——建筑设计可视化（8课时）⭐核心模块

| 课时 | 主题 |
|------|------|
| 第7课时 | 文生图基础——生成第一张建筑效果图 |
| 第8课时 | 建筑效果图提示词系统（风格/空间/渲染/光影词库） |
| 第9课时 | 室内设计AI可视化（多风格快速方案探索） |
| 第10课时 | 城乡规划与园林景观AI可视化 |
| 第11课时 | 图生图与AI修图（草图→效果图、SketchUp白模→渲染） |
| 第12课时 | 逻辑推理生图与高级技巧（一致性控制、多轮对话优化） |
| 第13课时 | 建筑安全演示图生成 |
| 第14课时 | 图像生成综合实训（全套可视化图纸 + 作品互评） |

---

### 模块四：音视频生成——方案展示（4课时）

**第15课时：音频生成**
- 文字转语音：项目汇报配音
- 双人播客：建筑知识讲解音频

**第16课时：文生视频与图生视频**
- 效果图 → 建筑漫游动画
- 首尾帧生成（工具：可灵、即梦、Runway）

**第17课时：高级视频生成**
- 数字人生成：虚拟讲解员
- 应用：项目路演、楼盘展示、文旅宣传

**第18课时：视频剪辑与整合**
- AI辅助剪辑 + 字幕配音 → 完整方案展示视频

### 模块五：PPT与汇报制作（2课时）

**第19课时：AI快速制作建筑方案PPT**

**第20课时：专业汇报PPT实战**
- AI生成PPT的二次优化技巧

## 五、课程思政主线："样式雷"文化融入

| 位置 | 融入方式 |
|------|---------|
| 模块一 | 以"样式雷"引出AI时代建筑设计的传承与创新 |
| 模块三 | 用AI重现/复原"样式雷"建筑图档，展示AI在文化遗产保护中的应用 |
| 模块六 | 制作"样式雷"文化知识智能体，让传统文化"活起来" |
| 贯穿全书 | 主线：**从样式雷到AIGC**——古有烫样呈现设计方案，今有AIGC数字化创新表达 |

> **申报优势**：独有文化资源+课程思政融合，是省级/国家级规划教材评审的显著亮点

## 六、每章统一结构

```
📖 章节导读（本章能学到什么，与建筑专业的关系）
📝 知识讲解（概念原理，简明扼要）
🖥️ 工具操作（分步骤截图演示）
🏗️ 建筑案例实操（4个专业方向各1个案例）
💡 技巧提示（常见问题与解决方法）
✏️ 课后练习（基础题 + 拓展题）
🔗 扫码拓展（二维码链接视频教程与资源更新）
```

四个专业方向案例覆盖：
- 🏘️ 城乡规划方向
- 🏢 建筑设计方向
- 🛋️ 室内设计方向
- 🌳 园林景观方向

## 八、AI+BIM学习路径（教师自学建议）

| 阶段 | 内容 | 时间 |
|------|------|------|
| 第一阶段 | 体验AI+建筑设计工具（小库科技、酷家乐） | 1-2周 |
| 第二阶段 | AI+SketchUp渲染工作流（白模→效果图） | 2-3周 |
| 第三阶段 | AI+Revit/BIM进阶（Autodesk Forma） | 按需 |
| 第四阶段 | 参数化设计+AI（Grasshopper+GhPython） | 选学 |

> 建议：不必等全部学完再加入教材。先在"未来设想"模块介绍AI+BIM，第二版修订时再增加独立章节。
