# 《建筑AIGC通用教程》十五五规划教材评审改进建议

> 以资深教材评审专家视角，针对省级/国家级十五五规划教材申报标准提出

---

## 一、当前教材优势（评审加分项）

| 维度 | 亮点 | 评审价值 |
|------|------|---------|
| **课程思政** | "样式雷"文化主线贯穿全书，学校是研究协会承办方 | 思政融入自然，有独特资源支撑，**强加分项** |
| **产教融合** | 四专业方向全覆盖，每章均有行业实操案例 | 体现"岗课赛证"融通理念 |
| **数字化配套** | 小程序+公众号+二维码更新体系 | 符合新形态教材建设要求 |
| **内容前沿性** | 工具矩阵更新至2026年，覆盖国产AI工具 | 体现科技自立自强 |
| **学习路径** | 文字→图像→视频→智能体→编程，渐进式设计 | 符合认知发展规律 |
| **市场定位** | 建筑行业AIGC教材市场空白，精准定位 | 填补空白，具有引领性 |

## 二、需要调整的方面（10项关键改进）

### A. 结构规范性问题（P0 优先）

#### 1. 缺少"学情分析"
**问题**：评审要求说明目标学生的知识基础、能力水平、学习特点
**建议**：在前言或教学指南中增加：
- 学生专业背景分析（建筑类高职，大二/大三）
- 计算机基础水平（基本办公软件，无编程基础）
- 学习特点（偏好实操，理论接受度需要案例支撑）
- 学习动机分析（就业竞争力提升，毕业设计效率）

#### 2. 缺少"教学设计说明"
**问题**：每个模块应有规范的教学设计表
**建议**：每模块增加教学设计表，格式如下：

```
| 教学目标层次 | 内容 | 对应布鲁姆认知目标 |
|-------------|------|-------------------|
| 了解 | AIGC基本概念和发展历程 | 记忆、理解 |
| 理解 | 文字生成和图像生成的基本原理 | 理解、分析 |
| 掌握 | 提示词七要素的编写方法 | 应用 |
| 应用 | 运用AI工具完成建筑专业文本写作 | 应用、创造 |

教学方法：讲授法 + 演示法 + 任务驱动法 + 小组合作法
评价方式：过程性评价（40%）+ 结果性评价（60%）
```

#### 3. 缺少"岗课赛证"对接说明
**问题**：高职教材评审核心指标之一
**建议**：增加岗课赛证融通矩阵表：

```
| 模块 | 对应岗位能力 | 对应赛项 | 对应证书 |
|------|------------|---------|---------|
| 模块一 | AI素养认知 | — | 1+X数字素养 |
| 模块二 | AI辅助文案撰写 | 全国职业技能大赛（建筑赛道方案文本） | — |
| 模块三 | AI辅助设计可视化 | 全国高校数字艺术设计大赛 | 1+X数字创意建模 |
| 模块四 | AI辅助方案展示 | 互联网+创新创业大赛 | — |
| ... | ... | ... | ... |
```

#### 4. 评价体系不够完善
**问题**：目前仅有课后练习，缺乏系统评价方案
**建议**：增加形成性评价方案：
- **过程性评价**（40%）：课堂参与、平时作业、实操记录
- **结果性评价**（40%）：作品集、综合实训项目
- **增值评价**（20%）：学期初/末AI素养自评对比
- 每模块增加评价量规（rubric）

### B. 内容深度问题（P1）

#### 5. 实践环节比重偏轻
**问题**：理论讲解篇幅大于实操步骤，高职教材应以实践为主
**建议**：
- 调整理论与实践比例为 3:7（目前约 5:5）
- 增加"分步操作指南"板块，配截图和操作说明
- 每个知识点后紧跟"动手试试"微实践

#### 6. 缺少"真实项目"案例
**问题**：现有案例多为模拟场景，评审看重真实性
**建议**：
- 至少增加1-2个校企合作真实项目案例
- 可以是教师参与的横向课题、学生实习项目
- 标注项目来源和合作单位（增加可信度）

#### 7. 缺少学生作品展示
**问题**：评审看重教学实效证据
**建议**：
- 开设试用班级，收集学生作品
- 精选优秀作品作为教材案例插图
- 增加"学生作品赏析"栏目

### C. 规划教材特色要求（P2）

#### 8. 缺少"活页式/工作手册式"设计元素
**问题**：十五五规划高职教材鼓励新形态教材
**建议**：
- 部分实训内容设计为可拆卸的工作页
- 例如：提示词编写练习工作页、AI输出评价工作页
- 工作页可独立使用，方便课堂分发和收集

#### 9. 课程思政需更系统化
**问题**：样式雷是好的切入点，但需要形成系统的"思政地图"
**建议**：
- 制作"课程思政融入一览表"
- 每个模块的思政点要有明确的育人目标
- 示例：

```
| 模块 | 思政主题 | 育人目标 | 融入方式 |
|------|---------|---------|---------|
| 模块一 | 科技自立自强 | 民族自信 | DeepSeek案例 |
| 模块一 | 文化传承 | 文化自信 | 样式雷引入 |
| 模块二 | 学术诚信 | 职业道德 | AI写作边界讨论 |
| 模块二 | 工匠精神 | 专业素养 | AI是工具不是专家 |
| 模块三 | 创新精神 | 创造力培养 | AI辅助设计探索 |
| ... | ... | ... | ... |
```

#### 10. 配套视频课程
**说明**：高质量配套视频是重要加分项，也是本次制作目标

## 四、评审答辩加分策略

### 4.1 差异化竞争力
1. **"样式雷"文化IP**：全国独有资源，思政融合自然
2. **AI做AI教材**：配套视频用AIGC制作，本身就是教材最好的实践证明
3. **建筑行业空白**：精准定位建筑类AIGC教材，填补市场空白
4. **数字化新形态**：小程序+公众号+视频+活页，多维度数字化配套

### 4.2 答辩展示建议
- 现场播放E01片头动画（15秒即可震撼）
- 展示"用AIGC技术制作AIGC教材视频"的闭环
- 强调样式雷文化融入的独特性
- 展示学生作品（如有试用班级数据）
