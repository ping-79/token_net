# 角色形象锚定提示词卡

> 用途：每次在即梦生成关键帧时，将对应角色的"锚定字符串"拼接到提示词最前面，保证跨镜头外观一致性。建议配合即梦"角色参考图"功能使用（先生成一张高质量参考图，后续图开启角色参考上传）。

## 四、统一视觉风格锚定（所有角色通用尾缀）

建议在每条提示词**末尾**追加下列统一风格串，保持10集视觉一致：

```
统一风格：2.5D皮克斯卡通渲染, 柔和轮廓线, 暖色主光+冷色补光, 电影级景深, 1920x1080构图, 建筑蓝#2E5C8A与暖橙#E8913A为主调
```

## 五、使用流程

1. **首次生成参考图**（每角色各一次）：
   - 在即梦"文生图"使用上方锚定字符串 + "正面全身像, 白色背景, T-pose参考"
   - 生成 4 张，挑最佳 1 张保存为 `ref_leigong.png` / `ref_xiaolei.png` / `ref_xiaozhu.png`
2. **后续分镜生成**：
   - 开启即梦"角色参考"，上传对应参考图
   - 提示词格式：`[角色锚定字符串] + [场景动作/表情/环境] + [统一风格尾缀]`
3. **一致性检查**：
   - 每 5 张抽检一次 checklist
   - 偏差明显则重生 or 在提示词前加强锚定词权重

## 六、三角色同框镜头的写法

当雷公+小雷同框（本集主戏）时，**两段锚定字符串顺序拼接**，并在场景描述里明确空间关系：

```
[雷公锚定字符串]。[小雷锚定字符串]。场景：现代建筑工作室内，雷公站在左侧惊讶地抬头看着悬浮的全息建筑模型，小雷站在右侧微笑地向他介绍，两人身高差明显（雷公略矮），中景构图。[统一风格尾缀]
```
