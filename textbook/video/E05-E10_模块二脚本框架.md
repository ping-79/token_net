## E05 学会和AI说话

**时长**：8-10分钟 | **即梦视频片段**：约8个 | **信息图/屏录**：约6个

### 内容大纲

| 时间段 | 内容 | 画面形式 | 旁白要点 |
|--------|------|---------|---------|
| 0:00-0:15 | 片头 | 标准片头动画 | — |
| 0:15-1:00 | 问题引入 | 小筑与AI对话的漫画场景 | "你跟AI说话，是不是经常觉得它答非所问？问题可能不在AI，而在你的提问方式" |
| 1:00-2:00 | 认识对话界面 | DeepSeek界面模拟图+标注 | 新建对话/连续追问/复制回答/重新生成四个要点 |
| 2:00-5:00 | 提问五层次（核心） | 分屏对比：左侧提问→右侧AI回答 | L1模糊→L2具体→L3场景→L4结构化→L5系统化，每层一个建筑案例 |
| 5:00-7:00 | 实操演示 | 屏幕录制+卡通指引叠加 | 场景1知识学习（解释容积率）+ 场景2规范解读 |
| 7:00-8:00 | 高效对话技巧 | 信息图卡片 | 角色指定/格式要求/多轮追问/关键原则 |
| 8:00-8:30 | 总结+课后任务 | 知识卡片+任务卡 | 用L4/L5提问完成一次建筑知识对话 |

### 重点场景提示词预留

**场景：小筑与AI答非所问（开场漫画）**
```
卡通漫画风格，两格分镜：第一格卡通女学生（戴安全帽）对着电脑输入"建筑设计怎么做？"，第二格电脑屏幕回复了一大段泛泛而谈的文字，女学生表情无奈头上冒汗，漫画对话气泡风格，幽默感
```

**场景：提问五层次阶梯图**
```
五级阶梯信息图，从左下到右上：L1模糊提问（灰色，最矮）→L2具体提问（浅蓝）→L3场景提问（蓝色）→L4结构化提问（深蓝）→L5系统化提问（金色，最高），每级旁边标注对应的AI回答质量星级，阶梯设计，渐变配色
```

**场景：L1 vs L4效果对比**
```
左右分屏对比，左侧标题"L1模糊提问"，对话框显示"建筑设计怎么做？"，下方AI回答模糊（灰色文字），右侧标题"L4结构化提问"，对话框显示完整的结构化提问，下方AI回答专业有条理（彩色高亮），对比鲜明，信息图风格
```

## E07 提示词七要素（核心课程）

**时长**：10-12分钟 | **即梦视频片段**：约12个 | **信息图动画**：约8个

### 内容大纲

| 时间段 | 内容 | 画面形式 | 旁白要点 |
|--------|------|---------|---------|
| 0:00-0:15 | 片头 | 标准片头动画 | — |
| 0:15-1:30 | 类比引入 | 建筑图纸→提示词类比动画 | "提示词就像设计图纸——图纸越详细，施工越精准" |
| 1:30-2:30 | 七要素总览 | 七个图标圆环+助记词动画 | "角背事要样风结"——七字口诀 |
| 2:30-4:00 | 要素1-2：角色+背景 | 对比演示（有角色vs无角色） | 室内设计案例：指定"20年经验设计总监"vs不指定 |
| 4:00-5:30 | 要素3-4：事项+要求 | 逐步构建提示词动画 | 城乡规划案例：明确任务+质量要求 |
| 5:30-7:00 | 要素5-6-7：样式+风格+结构 | 表格对比不同格式输出 | 园林景观案例：Markdown表格vs纯文本 |
| 7:00-8:30 | 综合实战 | 屏录：从零构建完整七要素提示词 | 建筑设计说明书完整案例 |
| 8:30-9:30 | 建筑专用模板库 | 四个模板卡片展示 | 设计说明/可研报告/分析报告/投标书模板 |
| 9:30-10:00 | 总结+课后任务 | 七要素速记卡 | 用七要素写一个建筑专业提示词 |

### 重点场景提示词预留

**场景：提示词=设计图纸（类比开场）**
```
左右对比画面，左侧是一张详细的建筑施工图纸（蓝图风格），右侧是一段格式化的提示词文本，两者之间用等号连接，底部标注"图纸越详细→施工越精准 | 提示词越完整→AI输出越专业"，信息图风格
```

**场景：七要素圆环图**
```
七个彩色圆形图标围成一个圆环，中央是"提示词七要素"文字，七个图标分别标注：角色（人形图标，红色）/背景（地图图标，橙色）/事项（任务图标，黄色）/要求（清单图标，绿色）/样式（格式图标，青色）/风格（画笔图标，蓝色）/结构（框架图标，紫色），底部显示助记词"角背事要样风结"，彩虹渐变配色
```

**场景：有角色vs无角色对比**
```
左右分屏对比，左侧标题"无角色指定"，提示框"请写一段客厅设计说明"，AI回答标注"一般化，像百科全书"（灰色），右侧标题"指定角色"，提示框"你是一位有20年经验的室内设计总监，请写..."，AI回答标注"专业、有经验感、有细节"（金色高亮），对比信息图
```

**场景：七要素逐步构建动画**
```
动画效果信息图，一个空白的提示词模板框，七个要素标签依次从右侧飞入并填入对应位置：角色→背景→事项→要求→样式→风格→结构，每填入一个要素，提示词框的"完成度进度条"增加一截，最终100%时提示词框发光，游戏化设计感
```
